# Coffee Machine Project 

A console-based **coffee vending machine simulation** in Python.  
Test your coding skills by managing resources, processing coins, and serving different types of coffee.

This project simulates a **coffee machine** that can serve **espresso, latte, and cappuccino**.  
The machine tracks its **resources** (water, milk, coffee) and **profit**.  

Key features:

- **Resource Management:** Checks if there are enough ingredients to make the requested drink.  
- **Coin Processing:** Accepts quarters, dimes, nickels, and pennies, calculates total payment, and gives change if necessary.  
- **Transaction Handling:** Verifies if the payment is sufficient, refunds if not, and updates profit.  
- **Reporting:** Displays current resources and total profit.  
- **Power Off:** Allows the machine to be turned off safely.  

This project helps practice **functions, loops, conditionals, and user input handling** in Python.
